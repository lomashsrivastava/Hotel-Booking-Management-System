# EcoStay Hotel Management System

A modern, eco-friendly Hotel Booking Management System designed to streamline hotel operations while promoting sustainability.

## Live Demo
[Live Preview](https://lomashbookingsystem.netlify.app/)

## Features
- **Interactive Dashboard**: Real-time overview of room status (Available, Booked, Maintenance).
- **Room Management**: Visual representation of rooms across multiple floors.
- **Booking System**: Easy-to-use form for customer check-ins and reservations.
- **Eco-Friendly Focus**: Special "Eco Mode" for energy-efficient room management.
- **Staff Controls**: Dedicated interface for staff to manage room status and maintenance.
- **Data Export**: Export booking data to Excel for record-keeping.
- **QR Code Integration**: Generate QR codes for easy payment or booking details.
- **Responsive Design**: Fully responsive layout that works on desktop and mobile devices.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Custom Properties, Flexbox, Grid), JavaScript (ES6+)
- **Libraries**:
  - Font Awesome (Icons)
  - Google Fonts (Orbitron, Exo 2)
  - Anime.js (Animations)
  - Chart.js (Data Visualization)
  - SheetJS (Excel Export)
  - QRCode.js (QR Code Generation)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
